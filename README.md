🔌 Snapper Chain Simulator

Google Code Jam 2010 Qualification Round Problem A「Snapper Chain（数珠繋ぎのスナッパー）」の動作を、ブラウザ上で視覚的に理解できるシミュレーターです。

スナッパー（スイッチ）の状態遷移や、電球が点灯するまでのプロセスをステップごとに確認できます。

✨ 特徴

視覚的なシミュレーション: 電流の流れとON/OFFの状態が一目でわかります。

履歴ログ機能: 指を鳴らした回数ごとの状態変化をリスト形式で記録・表示します。

自動実行モード: 電球が点灯する（すべてのビットが1になる）瞬間まで自動でステップを進めることができます。

レスポンシブデザイン: PCでもスマホでも快適に操作できます。

シングルファイル: すべてのコードが1つのHTMLファイルに含まれており、ダウンロードしてオフラインでも動作します。

🚀 使い方

スナッパーの数 (N) をスライダーで設定します（最大6個）。

「指を1回鳴らす」 ボタンを押すと、物理法則に従ってスナッパーのON/OFFが切り替わります。

「次の点灯まで自動実行」 を押すと、一番右の電球が光るまで連続して指を鳴らします。

「最初からやり直す」 でリセットできます。

📚 元ネタについて

Google Code Jam 2010 - Qualification Round Problem A: Snapper Chain

あなたはスナッパーと呼ばれる装置を持っています...
指をパチンと鳴らすと、電気が来ているスナッパーだけがON/OFFの状態を切り替えます...

この問題は、スナッパーの状態遷移が2進数（バイナリ）のカウントアップと同じ挙動をすることを利用して解くアルゴリズム問題として知られています。
